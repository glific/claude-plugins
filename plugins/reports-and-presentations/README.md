# Reports and Presentations

Skills for generating Glific's internal review decks.

| Skill | What it does |
|-------|-------------|
| `glific-monthly-review` | Generates the monthly review PPTX for any given month, pulling live data from Google Sheets and GitHub |
| `glific-quarterly-review` | *(coming soon)* Aggregates monthly data into a quarterly review deck |

## Setup

These skills require the Google Drive MCP to be connected (for reading source data and delivering the final file).
